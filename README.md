### Что такое область применения в javascript?
```
Область действия — это текущий контекст выполнения, в котором значения и выражения
являются «видимыми» или на них можно ссылаться. Если переменная или выражение отсутствует в текущем
scope, он не будет доступен для использования. Области также могут быть многоуровневыми в иерархии, 
Таким образом, дочерние области имеют доступ к родительским областям, но не наоборот.
```

# Что такое рекурсия в JavaScript?
## 03 Recursion
```js
Recursion — это когда функция вызывает себя до тех пор, пока кто-то ее не остановит. Если никто не остановит это, то это будет 
повторяться (называть себя) навсегда. Рекурсивные функции позволяют выполнять единицу работы несколько раз.
Современные языки программирования, такие как JavaScript, уже имеют операторы for ... while как
Альтернативы рекурсивным функциям. Но некоторые языки, такие как Closure, не имеют зацикливания
операторов, поэтому вам нужно использовать рекурсию для многократного выполнения фрагмента кода.
```
```js
Как только условие выполнено, функция перестает вызывать себя. Это называется базой 
состояние.
Чтобы предотвратить бесконечную рекурсию, вы можете использовать, if... else (или аналогичный подход) 
где одна ветвь делает рекурсивный вызов, а другая нет.

function factorial(n) {
    if (n == 0) {
        return 1
    }
    return n * factorial(n - 1)
}
console.log(factorial(5));
```
